# MRP Portfolio

Tracker de inversiones personal — acciones, cripto, CEDEARs, ETFs, bonos, tokenizadas, Trading Desk y sincronización en la nube.

## 🚀 Uso

Abrí la app directamente desde GitHub Pages:

**[👉 Abrir MRP Portfolio](https://criptoscrazy.github.io/mrp-portfolio-finance/)**


## ✨ Funcionalidades

- **Multi-activo**: Acciones, Cripto, CEDEARs, ETFs, Tokenizadas, Bonos
- **Precios en tiempo real** vía CoinGecko / Yahoo Finance
- **Trading Desk**: posiciones, diario de trading, checklist, calculadora de riesgo, watchlist, psicología, performance, reglas
- **Ingresos pasivos**: dividendos, staking, APY/APR con proyección automática
- **Dashboard**: IRR, Sharpe Ratio, Beta, distribución por custodio, comparador vs S&P 500 / BTC
- **Alertas de precio** y snapshots automáticos de evolución
- **Sincronización en la nube** vía Supabase + login con GitHub — accedé desde cualquier dispositivo
- **Modo oscuro / claro**, español / inglés
- **100% privado**: tus datos están protegidos con Row Level Security, solo vos podés verlos

## 🔒 Privacidad

Los datos se guardan en `localStorage` del navegador y, si conectás tu cuenta de GitHub, se sincronizan en una base de datos Supabase con seguridad a nivel de fila — nadie más que vos puede acceder a tu información.

## 🛠️ Stack técnico

Single-file HTML/CSS/JavaScript. Sin build, sin dependencias de instalación. Chart.js para gráficos, Supabase para autenticación y sincronización.

## 📱 Instalar como app

Podés agregar la página a la pantalla de inicio de tu iPhone o como acceso directo de escritorio para que se sienta como una app nativa.
